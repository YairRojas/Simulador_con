Presentación

El nombre del proyecto es "conservación de energía", se inspira en el primer simulador qu se ha realizado en la clase pero este tendrá una peculiaridad en que la masa que uno podrá digitar sería infinita y una pista más amplia y moldearlo de maneras más diversas. Este proyecto lo tenemos pensado realizarlo para página web vamos a usar los lenguajes HTML5, CSS3 y JavaScript. Integrantes del grupo: Laura Tovar: Realizará el contenido visual como, la creación de las imágenes que se usarán en el proyecto. Yair Rojas: Realizará la parte visual de la aplicación web que muestra el personaje, la pista, el entorno, etc. Luis Rodriguez: Realizará la parte lógica y funcional de la web como el mover la posición del personaje, la gravedad, el movimiento al soltar, etc.
